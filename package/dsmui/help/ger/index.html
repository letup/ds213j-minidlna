MiniDLNA 3rd Party Paket

Distribution des MiniDLNA Medienservers.

MiniDLNA (aka ReadyDLNA) strebt die vollständige Unterstützung DLNA/UPnP-AV fähiger Abspielgeräte an. Die Software wird von einem NETGEAR Mitarbeiter für die ReadyNAS Modellreihe entwickelt.

Die Installation und Nutzung dieses 3rd party Pakets geschieht auf eigene Gefahr. Beachte bitte den allgemeinen Haftungsausschluss im Synology Wiki.
