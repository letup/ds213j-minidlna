MiniDLNA 3rd Party Package

A distribution of MiniDLNA UPnP-AV media server.

MiniDLNA (aka ReadyDLNA) is server software with the aim of being fully compliant with DLNA/UPnP-AV clients. It is developed by a NETGEAR employee for the ReadyNAS product line.

Installation and usage of this 3rd party package is at your own risk. Please read Synology's General Disclaimer.
